# Anti-Diagonal Traversal of a Matrix

## Problem Statement
Give a n * n square matrix mat[][], return all the elements of its anti-diagonals from top to bottom.

---

## Example

### Input: 
```
n = 3
mat[][] = [
        [1, 2, 3],
        [4, 5, 6],
        [7, 8, 9]]
```
### Output: 
` [1, 2, 4, 3, 5, 7, 6, 8, 9] `
